# Лабораторная работа №1: Настройка окружения разработки

## Описание
Данная лабораторная работа посвящена настройке базового окружения для разработки на Python и работе с системой контроля версий Git.

## Выполненные задачи

В рамках лабораторной работы были настроены следующие инструменты:

- **VS Code** — текстовый редактор с поддержкой расширений для Python
- **Python 3.x** — интерпретатор языка программирования
- **Git** — система контроля версий для управления кодом
- **GitHub** — платформа для хостинга репозиториев

## Пример кода

Простая программа на Python для демонстрации базового синтаксиса:
```python
def greet(name):
    """
    Функция приветствия пользователя
    """
    return f"Привет, {name}! Добро пожаловать в мир программирования!"

# Основная программа
if __name__ == "__main__":
    user_name = input("Введите ваше имя: ")
    message = greet(user_name)
    print(message)
```
